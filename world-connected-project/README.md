# 🌍 World Connected Project

## 📌 Overview

The **World Connected Project** is a data analysis project that explores the relationship between **global population growth** and **internet adoption** over time. Using historical population datasets and internet usage statistics, this project analyzes how digital connectivity has expanded across different continents and countries.

The project aims to uncover trends in internet penetration, population growth, and the pace at which the world is becoming digitally connected.

---

## 🎯 Project Objectives

* Analyze historical world population trends
* Study growth in global internet users
* Compare internet adoption across continents
* Identify regions with fastest digital growth
* Visualize trends using charts and graphs
* Derive insights about global connectivity

---

## 📂 Dataset Information

This project uses multiple datasets from trusted public sources:

### Population Data

* Historical world population data
* Population by continent over time

### Internet Usage Data

* Global internet users dataset
* Internet users by continent
* Historical internet adoption statistics

### Data Sources

* **Our World in Data**
* **Gapminder**
* **United Nations (UN WPP)**
* **Statista**

## 🔍 Key Insights

* Global internet users have grown exponentially in the last two decades.
* Developed continents show higher internet penetration.
* Population growth and internet adoption do not always increase at the same rate.
* Emerging economies are experiencing rapid digital transformation.
* Asia contributes the largest share of global internet users due to population size.
